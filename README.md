# binance-batch-dwh-
binance-batch-dwh/


Какие данные храним: OHLCV свечи:
- Open (Цена открытия)
- High (Максимум за период)
- Low (Минимум за период)
- Close (Цена закрытия)
- Volume (Объем торгов)

Режим: Batch 
Переодичность: 1 час / 1 день


Цель проекта: Аналитика и BI 

Перспектива: датасет, используемый в дальнейшем в реализации RL-агента для алгоритмической торговли на Binance.
